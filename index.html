COMP 1950

This hands-on course follows on from COMP 1850 Applied Web Development Level 1.

Students who already understand HTML and CSS will explore web development in more depth with new topics and techniques. Participants will gain a deeper understanding of the latest technologies including HTML5 and CSS3 for developing web sites and applications. There is a focus on responsive design and mobile presentation through the use of Media Queries. Adding behavior to web sites using JavaScript frameworks such as jQuery is introduced. Labs and exercises focus on industry standard web development methodology.

COMP 1950 is a required course for the BCIT Computing Advanced Web Technologies associate certificates in Applied Web Development, AWD and Web and Mobile Application Development, WMA, as well as an elective in the CST Diploma. COMP 1950 and COMP 2015 are also the prerequisites for COMP 2052 Android and Mobile Application Development Tools. By the end of this course successful students will be able to build dynamic and semantically marked-up content providing Search Engine Optimized (SEO) pages.

Course Outcomes

  1. Mark up pages using latest HTML technologies, including HTML5.
  2. Utilize CSS to control page layout and presentation, including
  3. CSS3.
  4. Use media queries to create Responsive designs for mobile and desktop presentation.
  5. Add behaviour to any web page using JavaScript.
  6. Create dynamic content using JavaScript frameworks like jQuery.
  7. Employ W3C Web Standards when building web sites for maximum compatibility.
  8. Understand concepts for Search Engine Optimized content (SEO).
  9. Install, configure and understand server side architecture using Apache.
  10. Understand template based front end development using Bootstrap and HTML Boilerplate.
  11. Understand team-based web development methodology.
  12. Use HTML5 to present rich content such as video, audio and geolocation.
  13. Create professional looking typography using web-based fonts.
  14. Understand source control using GIT.
